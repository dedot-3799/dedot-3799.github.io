The Risk of JavaScript

// JavaScript
function add(a, b) { return a + b; } console.log(add(10, "20")); // 1020 (Oops, that's not a number!)

Are you OK with this happening? TypeScript catches this mistake instantly.

The Safety of TypeScript

// TypeScript
function add(a: number, b: number): number { return a + b; } console.log(add(10, "20")); // Error! You're protected from this!

TypeScript prevents this bug before it even happens. No more worrying about runtime errors.
